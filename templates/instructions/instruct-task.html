To navigate through the room and pick/place items, you can use the following keyboard controls:

  • W: To move forward
  • Space: Pick up an item/place the picked item back

You can look around using the following keyboard controls:

  • Left arrow/A: Turn left
  • Right arrow/D: Turn right
  • Up arrow: Look up
  • Down arrow: Look down

Note that you canâ€™t walk through walls, furniture or objects.

To pick up any object you have to point the crosshair which is at the center of the camera to the object you want to grab. The items you pick up and are a part of the agent body. Note that you can only carry one object at a time. If you try to grab a second object simultaenously the first object which was picked up will be dropped. Also note that objects not highlighted are not pickable. You have to be close enough to an object to pick it up. Try moving closer to the object if you are unable to pick it up. Objects can be dropped anywhere in the scene where there is no collision with another highlighted. Sometimes, looking down at the bin also helps while placing the item!
