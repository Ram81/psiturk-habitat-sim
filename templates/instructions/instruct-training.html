Instructions

This is the second stage of this HIT. This is your chance to become acquainted with the keyboard controls and the environment! For this, complete the task instruction on top of the environment screen.

To navigate through the room and pick/place items, you can use the following keyboard controls:

  • W: To move forward
  • S: To move backward
  • Spacebar: Pick up an item/place the picked item back

You can look around using the following keyboard controls:

  • Left arrow/A: Turn left
  • Right arrow/D: Turn right
  • Up arrow: Look up
  • Down arrow: Look down

Please note that you canâ€™t walk through walls, furniture or objects.

When you pick up an item to carry it to another location, it will be displayed in your inventory menu at the bottom of the screen. Note that you can only carry one item at a time!

To pick up any object you have to point the crosshair which is at the center of the camera to the object you want to grab. If you are within 1.5m of the object you'll see a bounding box around the object which indicates that you can grab the object. If you try to grab a second object simultaenously the first object which was picked up will be dropped. Also note that objects not highlighted are not pickable. You have to be close enough to an object to be able to pick it up. Try moving closer to the object if you are unable to pick it up. If you point the crosshair at a location within 1.5m radius of agent position the object will be placed at the location pointed by crosshair. Otherwise the object will be dropped 1m in front of the agent position from a height of 1.5m.

The task is given as a instruction at the top of the experiment page.
