Instructions

To navigate through the room and pick/place items, you can use the following keyboard controls:

  • W: To move forward
  • S: To move backward
  • Spacebar: Pick/Place an object

You can look around using the following keyboard controls:

  • Left arrow/A: Turn left
  • Right arrow/D: Turn right
  • Up arrow: Look up
  • Down arrow: Look down

Instructions when interacting with objects:

  • When you pick up an object to carry it to another location, it will be displayed in your inventory menu at the bottom of the screen.
  • You can only carry one object at a time.
  • To pick up any object you have to aim the cursor to the object you want to pick and hit the Spacebar. If you are within 1.5m of the object you'll see a box around the object which indicates that you can pick up the object.
  • If you try to pick a second object the first object which was picked up will be dropped.
  • To place an object, walk to within 1.5m of the target location, aim your cursor at the point where you want to place the object and hit Spacebar.
